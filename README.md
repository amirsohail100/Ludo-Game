## 🌟 Key Features

- **Pure JavaScript Logic:** Independent game loop and token movement calculations without heavy external frameworks.
- **Responsive Layout:** Designed to scale beautifully across mobile devices, tablets, and desktops (Work in Progress).
- **Interactive Dice:** Pseudo-random dice rolling mechanics with CSS animations.
- **Modern UI Styling:** Clean, vibrant, and minimalist UI designed using advanced CSS techniques.

---

## 🛠️ Tech Stack

- **HTML5:** Structures the Ludo board layout, player zones, and control panels.
- **CSS3:** Powers the board styling, grid alignment, responsive scaling, and token animations.
- **JavaScript (ES6):** Handles the core game state, player turns, rolling logic, and win-conditions.
